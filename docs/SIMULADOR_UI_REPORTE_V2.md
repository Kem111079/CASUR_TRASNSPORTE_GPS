# Simulador V2 · UI y reporte ejecutivo

## Objetivo
Hacer la demostración más inmersiva y más vendible para gerencia/logística.

## Cambios clave
- Bitácora móvil de 3 niveles para no competir con el mapa.
- Mini barra operativa durante recorrido.
- Mapa de reporte con fondo operativo vectorial, ruta, flechas, inicio, fin y paradas.
- Reporte compacto por tramos para evitar documentos largos en viajes extensos.
- Excel mantiene detalle técnico completo y agrega hoja Tramos.

## Lógica de compactación
El PDF/HTML no lista todos los puntos ni todos los lotes cercanos. El reporte ejecutivo muestra:
- máximo de tramos relevantes;
- paradas principales;
- lugares principales;
- nota de referencias adicionales compactadas.

El detalle completo permanece en Excel.
